# Hello-World

### Welcome To Programming

Welcome to the world of Programming! <br />

In this challenge, We will learn how to print “Hello World!” in standard output (stdout). <br />

Input Format: <br />
There is no input for this challenge. <br />

Output Format: <br />
You must print two lines of output:

Print "Hello, World." on the first line (without quotes). <br />

Print "Hello, Java." on the second line (without quotes). <br />

Sample Output: <br />
Hello, World. <br />
Hello, Java.
